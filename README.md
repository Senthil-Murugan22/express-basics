# Express.js Basics 🚀

This repository contains my beginner practice using Express.js.

## 📌 Topics Covered

* Basic routing (/, /about, /contact)
* Sending JSON responses (API)
* Query parameters (req.query)
* Dynamic user input handling

## ▶️ How to Run

Install dependencies:

```bash
npm install express
```

Run any file:

```bash
node basic-routes.js
```

## 🌐 Example URLs

* http://localhost:3000
* http://localhost:3000/about
* http://localhost:3000/api/user
* http://localhost:3000/api/user?name=SENTHIL&age=22
* http://localhost:3000/api/greet?name=SENTHIL
